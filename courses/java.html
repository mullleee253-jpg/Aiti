Урок 1: Введение в Java

Что такое Java?

Java - это объектно-ориентированный язык программирования, который используется для:

  • Enterprise приложений
  • Android разработки
  • Веб-приложений
  • Игр (Minecraft написан на Java!)

Первая программа

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Привет, мир!");
    }
}

Структура программы

  • public class - объявление класса
  • main - точка входа в программу
  • System.out.println() - вывод на экран

Создайте программу, которая выводит ваше имя.

Урок 2: Переменные и типы данных

Типы данных в Java

int age = 25;           // Целое число
double price = 99.99;   // Дробное число
String name = "Иван";   // Строка
boolean isActive = true; // Логическое значение
char grade = 'A';       // Символ

Константы

final double PI = 3.14159;
final int MAX_SIZE = 100;
