Урок 1: Введение в C++

Что такое C++?

C++ - мощный язык программирования для:

  • Разработки игр (Unreal Engine)
  • Системного программирования
  • Высокопроизводительных приложений
  • Embedded систем

Первая программа

#include <iostream>
using namespace std;

int main() {
    cout << "Привет, мир!" << endl;
    return 0;
}

Создайте программу, которая выводит ваше имя.
